# Araç Kiralama Otomasyonu

Bu proje, araç kiralama süreçlerini dijitalleştirmek ve kolaylaştırmak amacıyla geliştirilmiş bir masaüstü uygulamasıdır. Kullanıcı dostu arayüzü sayesinde müşteri kayıtları, araç müsaitlik durumları ve kiralama işlemleri uçtan uca yönetilebilmektedir.

# Kullanılan Teknolojiler
* **Programlama Dili:** Python
* **Kullanıcı Arayüzü (GUI):** Tkinter
* **Veri Yönetimi:** JSON (Veri Saklama ve Okuma)

## Temel Özellikler
* **Araç ve Müsaitlik Yönetimi:** Sistemdeki araçların durumlarının (kirada/müsait) güncel olarak takip edilmesi.
* **Müşteri Kayıt Sistemi:** Kiralama işlemi yapan müşterilerin temel bilgilerinin alınması ve sisteme işlenmesi.
* **Kiralama Mantığı (Business Logic):** Seçilen araca ve kiralama süresine göre işlemlerin gerçekleştirilmesi.
* **Etkileşimli Arayüz:** Tkinter kütüphanesi ile baştan sona ortaklaşa tasarlanmış, kullanımı kolay grafiksel arayüz.

## Kurulum ve Çalıştırma
Projeyi kendi yerel ortamınızda çalıştırmak için aşağıdaki adımları izleyebilirsiniz:

1. Bu repoyu bilgisayarınıza klonlayın:
   git clone https://github.com/tunahankavakli/arac-kiralama-otomasyonu.git
   
2. Proje dizinine gidin:
   cd arac-kiralama-otomasyonu
   
3. Ana uygulamayı çalıştırın:
   python main.py


*Bu proje, 2 kişilik bir geliştirici takımı ile ortaklaşa kodlanmıştır.*
